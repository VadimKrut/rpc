# rpc-contract

`rpc-contract` defines immutable RPC method and service metadata.

## Use it when

- you want explicit method contracts
- you want service-level duplicate checks before runtime
- you build higher-level client or server wiring

## Dependency

```xml
<dependency>
    <groupId>ru.pathcreator.pyc</groupId>
    <artifactId>contract</artifactId>
    <version>0.0.3-SNAPSHOT</version>
</dependency>
```
