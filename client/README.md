# rpc-client

`rpc-client` is the typed client-side layer built on top of `rpc-core` and `rpc-contract`.

## Features

- typed request/response calls
- bound calls
- interceptors and validators
- structured remote error handling
- timeout-aware request lifecycle

## Dependency

```xml
<dependency>
    <groupId>ru.pathcreator.pyc</groupId>
    <artifactId>client</artifactId>
    <version>0.0.1</version>
</dependency>
```
