Реестр сервисов

Сервисный слой — единственное место, где доступны операции включения и отключения отдельных методов.

Сервис Размещение Канал Состояние Методы Операция
vpn.gateway vpn-environment/vpn-server channel:1 Включен 1

Серверные контуры

Серверная метрика латентности показывает только обработку на стороне исполнителя, без клиентского round-trip.

Сервер Размещение Канал Состояние Запросы Операция
vpn-server vpn-environment/vpn-server channel:1 Включен 0

Клиентские контуры

Клиентская латентность включает публикацию запроса, ожидание ответа и полное время round-trip.

Клиент Размещение Канал Состояние Вызовы Операция
vpn-client vpn-environment/vpn-client channel:2 Включен 0
