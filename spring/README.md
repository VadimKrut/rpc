# rpc-spring

`rpc-spring` integrates the bootstrap layer with Spring Boot auto-configuration.

## Features

- `@RpcEndpoint`
- automatic client proxy registration
- automatic endpoint export
- `application.yml` driven channel profiles
- optional encryption wiring from properties

## Dependency

```xml
<dependency>
    <groupId>ru.pathcreator.pyc</groupId>
    <artifactId>spring</artifactId>
    <version>0.0.2-SNAPSHOT</version>
</dependency>
```
