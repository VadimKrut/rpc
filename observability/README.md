# rpc-observability

`rpc-observability` provides optional low-overhead listeners and metrics snapshots for clients and servers.

## Features

- call counters
- latency snapshots
- status-based error counters
- timeout accounting
- bytes and in-flight tracking

## Dependency

```xml
<dependency>
    <groupId>ru.pathcreator.pyc</groupId>
    <artifactId>observability</artifactId>
    <version>0.0.1</version>
</dependency>
```
