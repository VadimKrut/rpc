# rpc-server

`rpc-server` is the typed server-side dispatcher built on top of `rpc-core` and `rpc-contract`.

## Features

- method registration by contract
- context-aware handlers
- interceptors and validators
- status-aware error mapping
- optional payload encryption integration

## Dependency

```xml
<dependency>
    <groupId>ru.pathcreator.pyc</groupId>
    <artifactId>server</artifactId>
    <version>0.0.3-SNAPSHOT</version>
</dependency>
```
